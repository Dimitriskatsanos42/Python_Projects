# Python Projects 🐍

Ένα ανοιχτό "σκαλί" μικρών Python projects — αλγόριθμοι, παιχνίδια, και διάφορα mini projects που έφτιαξα καθώς μάθαινα και εξασκούμουν στη γλώσσα.

Αν είσαι κι εσύ στην αρχή με την Python, ρίξε μια ματιά — ίσως βρεις κάτι που σε εμπνεύσει ή σε βοηθήσει να ξεκινήσεις κάτι δικό σου. Η ιδέα είναι απλή: μικρά, αυτοτελή κομμάτια κώδικα, το καθένα να δείχνει κάτι διαφορετικό που μπορεί να κάνει η Python.

## Τι θα βρεις εδώ

**`Algorithms/`**
Υλοποιήσεις αλγοριθμικών προβλημάτων — δομές δεδομένων, ταξινομήσεις, αναζητήσεις, ό,τι έχει να κάνει με λογική και πολυπλοκότητα.

**`Python_Games/`**
Μικρά παιχνίδια φτιαγμένα σε Python — καλή άσκηση για λογική, βρόχους, και διαχείριση κατάστασης παιχνιδιού.

**`Python_Mini_Projects/`**
Μια συλλογή από ανεξάρτητα mini projects — μικρά εργαλεία, scripts, ή ασκήσεις που δεν χωράνε αλλού.

**`Maximum_Subarray_Sum.py`**
Ένα αυτόνομο script στη ρίζα του repo — υλοποίηση του κλασικού προβλήματος Maximum Subarray (πιθανότατα με το αλγόριθμο του Kadane).

## Γιατί το κράτησα έτσι, χωρίς αυστηρή δομή

Επίτηδες δεν είναι οργανωμένο σαν "epic" project με ενιαία αρχιτεκτονική — είναι περισσότερο ένα defter, ένα μέρος όπου μαζεύονται μικρά πειράματα. Κάθε φάκελος/αρχείο μπορεί να ανοιχτεί και να τρέξει ανεξάρτητα, χωρίς να χρειάζεται να καταλάβεις τα υπόλοιπα.

## Πώς να τρέξεις κάτι από εδώ

```bash
git clone https://github.com/Dimitriskatsanos42/Python_Projects.git
cd Python_Projects
```

Για ένα μεμονωμένο script:

```bash
python Maximum_Subarray_Sum.py
```

Για κάτι μέσα σε φάκελο:

```bash
cd Python_Games
python <όνομα_αρχείου>.py
```

Οι περισσότερα scripts δεν χρειάζονται εξωτερικές βιβλιοθήκες — αν κάποιο χρειάζεται (π.χ. `pygame` για κάποιο παιχνίδι), θα το δεις σε `import` στην αρχή του αρχείου. Εγκατάστησέ το με:

```bash
pip install <όνομα_βιβλιοθήκης>
```

## Τι εξασκεί αυτό το repo

Λίγη λογική, λίγη Python syntax, και πάνω απ' όλα τη συνήθεια να παίρνεις μια ιδέα και να τη μετατρέπεις σε λειτουργικό κώδικα — χωρίς να χρειάζεται να είναι τέλειο ή μεγάλο για να αξίζει να υπάρχει.
